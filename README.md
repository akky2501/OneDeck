# OneDeck
Game Description Language for Playing Cards

このプロジェクトは[株式会社なうデータ研究所](https://www.nau.co.jp/)様のインターン成果です．
(インターン期間 8/13 - 8/23，9/5)

## What
- トランプを用いたカードゲームとプレイヤーの行動選択を記述するための言語
- ゲーム進行のナビゲーションシステム

詳細はdocument内の資料を参照
